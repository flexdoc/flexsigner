Para instalar o serviço:

installutil FlexSignerService.exe  


Se você estiver usando o prompt de comando do Visual Studio, 
o InstallUtil.exe deve estar no caminho do sistema. 
Caso contrário, você pode adicioná-lo ao caminho ou usar o 
caminho totalmente qualificado para invocá-lo. 
Essa ferramenta é instalada com o .NET Framework e o caminho 
é %WINDIR%\Microsoft.NET\Framework[64]\<framework_version>. 
Por exemplo, para a versão de 32 bits do .NET Framework 4 ou 4.5.*,
se o diretório de instalação do Windows for C:\Windows, 
o caminho é C:\Windows\Microsoft.NET\Framework\v4.0.30319\InstallUtil.exe. 

Para a versão de 64 bits do .NET Framework 4 ou 4.5.*, o caminho padrão
é C:\Windows\Microsoft.NET\Framework64\v4.0.30319\InstallUtil.exe.


Obs.: Usualmente> C:\Windows\Microsoft.NET\Framework64\v4.0.30319
